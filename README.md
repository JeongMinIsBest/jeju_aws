# 🌊 제주 연안 고수온 리스크 지수 (HRI) 산출 파이프라인
위성 기반 해수면 수온(SST) 데이터를 활용하여 제주 연안의 고수온 위험을 정량화하고,  
보험 · 정책 · 조기경보 트리거로 활용 가능한 Heat Risk Index(HRI)를 산출하는 파이프라인입니다.
<br/>
<br/>

## 🎯 프로젝트 목표
- 위성 SST 기반 **일 단위 고수온 리스크 지수(HRI)** 산출
- 정책 · 보험에 바로 적용 가능한 **정량적 트리거 구조** 제시
- “고수온 특보” 기준과 **정합되는 데이터 파이프라인 구축**
<br/>
<br/>

## 🛰 사용 데이터

### 1️⃣ 위성 해수면 수온 (SST)
- **데이터명** : OISST (Optimum Interpolation Sea Surface Temperature)
- **특징**
  - 전 지구 일 단위 SST 제공
  - 위성 · 부이 · 관측 자료 보정 결합
  - 공공 · 표준 데이터
- **공간 범위** : 제주 연안 인근 격자 평균
- **시간 범위** : 사용자 설정 (예 : 2024-07 ~ 2024-10)
<br/>

### 2️⃣ 기준 정책 데이터

- 해양수산부 / 기상청 고수온 특보 기준
  - `28°C 이상, 72시간 지속` 등
<br/>
<br/>

## 🧮 HRI (Heat Risk Index) 개념

HRI는 단순 온도 초과 여부가 아니라 기준 임계 온도 초과 정도, 누적 지속성, 단기 변동성을 종합적으로 반영한 **연속형 리스크 지수**입니다.
<br/>
<br/>
